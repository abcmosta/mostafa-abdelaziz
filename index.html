Selected Work & Process

ERDR DPMO Optimization

530K ➔ 285K DPMO

The Problem

Virtual-physical mismatches on expiry records in Amazon's AFT system due to vendor non-compliance (mixed expirations) and human error at receive stations.

My Process

Existing software couldn't pinpoint root causes. I bypassed it by designing an Excel architecture 100% reliant on custom SQL database queries to extract exact defect injection points, ensuring a strictly data-driven action plan.

The Solution

Implemented proactive floor practices, established strict expiry audits, and scheduled random strange-area audits on expirable products to intercept defects before they impacted the DPMO metric.

IRDR Metric Control

26% ➔ 3% Defects

The Problem

High Inventory Records Defect Rate (IRDR) where physical warehouse quantities failed to match virtual system records, causing fulfillment delays.

My Process

I engineered 3 different ETL (Extract, Transform, Load) jobs with multiple SQL queries. This allowed me to track the exact traffic and associate behaviors causing inventory deviations.

The Solution

Targeted the root cause by scheduling automated, routine cycle counts specifically directed at the highly-defective bins identified by the ETL pipelines.
